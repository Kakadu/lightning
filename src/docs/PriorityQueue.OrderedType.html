Module type PriorityQueue.OrderedType


module type OrderedType = sig .. end
type t 
val order : t -> t -> bool
The type of priority orders on 'a. A priority order ord is a non-strict total order where ord a b = true means that the priority of b is not higher than the priority of a. When the priority of an element changes, the queue must be notified using the reorder_up or reorder_down function.
